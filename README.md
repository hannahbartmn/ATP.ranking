
# Installation

The package can be installed with:

``` r
library(devtools)

install_github("hannahbartmn/ATP.ranking")
```

# Contents

Find out about the dataset and the weight functions with:

``` r
help(package = "ATP.ranking")
```

# Dataset

The dataset can be accessed with:

``` r
data("tennis_data")
```
